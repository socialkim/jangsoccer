# 장르만 여의도: 마우스복싱 FC 3D

JTBC 《장르만 여의도》의 가벼운 정치 토크와 논점 대결 분위기를 5대5 아케이드 축구로 재해석한 비공식 브라우저 게임 콘셉트입니다.

> 본 프로젝트는 JTBC 공식 게임이 아닙니다. 인물별 포지션과 특수기는 프로그램 포맷을 게임 문법으로 각색한 창작 설정입니다.

## 바로 실행

GitHub Pages 배포 주소:

- https://socialkim.github.io/jangsoccer/

PC와 모바일 브라우저에서 별도 설치 없이 실행할 수 있습니다. 모바일에서는 홈 화면에 추가해 PWA처럼 실행할 수도 있습니다.

## 주요 기능

- 5대5 3D 아케이드 축구와 AI 골키퍼
- 득점 → 리플레이 → 3·2·1 재킥오프 → 자동 경기 재개
- 90초, 120초, 180초 경기 시간
- 쉬움, 보통, 어려움 AI 난이도
- 패스, 태클, 질주, 강슛, 선수 전환
- 출연자별 특수 능력
- 추적·중계·전술 카메라
- FACT, ENERGY, +5 SEC 논점 토큰
- 슈팅, 패스, 점유율 실시간 통계
- 경기장 관중음, 킥, 휘슬, 골, 특수기 효과음
- 서울 야경 이미지와 CC0 외부 UI 사운드
- 모바일 터치 조작, 진동 피드백, 노치·안전영역 대응
- 모바일 성능을 위한 낮은 렌더 해상도와 관중 오브젝트 최적화
- 서비스 워커 기반 핵심 리소스 오프라인 캐시

## 조작법

### PC

| 기능 | 키 |
|---|---|
| 이동 | `WASD` 또는 방향키 |
| 질주 | `Shift` |
| 슛 | `Space` |
| 패스 / 태클 | `F` |
| 캐릭터 특수기 | `E` |
| 선수 전환 | `Q` |
| 카메라 전환 | `C` |
| 음소거 | `M` |
| 일시정지 | `P` 또는 `Esc` |

### 모바일

- 왼쪽 방향 패드: 이동
- `BOOST`: 누르고 있는 동안 질주
- `KICK`: 슛
- `F`: 패스 또는 태클
- `E`: 특수기
- `Q`: 선수 전환
- 우측 카메라 버튼: 카메라 전환
- 우측 일시정지 버튼: 경기 중단

세로 화면에서도 플레이할 수 있지만 가로 화면에서 경기장을 더 넓게 볼 수 있습니다.

## 모바일 점검 결과

Chromium 모바일 에뮬레이션 환경에서 390×844 화면을 기준으로 다음 항목을 확인했습니다.

- 메뉴 내부 스크롤과 경기 시작 버튼 접근
- 메뉴에서는 터치 컨트롤 숨김, 경기 중 자동 표시
- 좌우 터치 입력 방향 정상
- 질주·카메라·일시정지 터치 버튼 이벤트
- 득점 후 `goal → kickoff → playing` 상태 복귀
- 점수 유지 및 재킥오프
- 미니맵과 모바일 보조 버튼 비겹침
- 모바일 렌더 캔버스 크기 최적화

실제 기기 성능과 브라우저 설정에 따라 프레임 속도는 달라질 수 있습니다. 최신 Chrome, Samsung Internet, Safari 사용을 권장합니다.

## 로컬 실행

```bash
python -m http.server 8000
```

브라우저에서 `http://localhost:8000`으로 접속합니다. 브라우저 정책상 소리는 경기 시작 버튼을 누른 뒤 재생됩니다.

## 자동 배포

`.github/workflows/pages.yml`이 `main` 브랜치에 푸시될 때 정적 파일 전체를 GitHub Pages로 배포합니다.

GitHub 저장소에서 Pages가 아직 활성화되지 않았다면 최초 한 번만 다음 항목을 확인해야 할 수 있습니다.

1. `Settings → Pages`
2. `Build and deployment → Source`
3. `GitHub Actions` 선택

이후에는 `main` 브랜치 변경 사항이 자동으로 배포됩니다.
