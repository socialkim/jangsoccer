## Kenney Interface Sounds

The game requests the following UI sounds from the public GitHub mirror of Kenney Interface Sounds:

- `confirmation_001.wav`
- `click_001.wav`
- `switch_001.wav`
- `error_001.wav`

The Kenney Interface Sounds pack is released under CC0. Network failure falls back to locally generated Web Audio tones, so gameplay does not stop when the remote files are unavailable.
